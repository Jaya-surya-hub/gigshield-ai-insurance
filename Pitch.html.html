CORE INNOVATION

IFI — Same Rain,
Different Reality

Traditional insurance uses one rainfall threshold for an entire city. But a well-drained hilltop and a low-lying floodplain react completely differently to the same rain.

Our Gemini AI agent performs autonomous web research on every 1km² zone — civic tenders, drain capacity, recent pothole reports — and scores each zone's Infrastructure Fragility Index (IFI).

ANTI-FRAUD ENGINE

We Trust Physics,
Not GPS

GPS coordinates can be spoofed with a $2 app. But kinematic variance from IMU hardware sensors — accelerometer, gyroscope — cannot be faked.

Our IsolationForest model analyzes claim frequency, distance from zone, GigScore history, and real-time sensor data to flag anomalies instantly.
